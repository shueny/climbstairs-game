# 爬格子小遊戲
## Demo
[https://shueny.github.io/climbstairs-game/](https://shueny.github.io/climbstairs-game/)

## 簡介
依照 input 所輸入的數字自動產生爬格子的直線，以及亂數產生橫線的節點並產生橫線。點選 MOMOCO 大頭後，會自動爬格子

## 技術：
- jQuery
- Array
- SVG
